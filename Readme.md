# ☕ Coffee Shop Sales Analysis

## 📌 Project Overview

Developed an interactive Coffee Shop Sales Dashboard in Microsoft Excel to analyze sales performance, customer footfall, product trends, and store performance. The dashboard helps identify business insights and supports data-driven decision-making.

## 📊 Key Metrics

* Total Sales: $698,812
* Total Footfall: 149,116
* Average Bill per Person: $4.69
* Average Orders per Person: 1.44

## 🧹 Data Cleaning

* Removed duplicate records
* Checked and handled missing values
* Standardized date and time formats
* Validated data accuracy
* Prepared data for analysis

## 📈 Analysis Performed

* Sales Analysis
* Customer Footfall Analysis
* Store-wise Performance Analysis
* Product-wise Sales Analysis
* Category-wise Sales Distribution
* Hourly Sales Trend Analysis
* Weekday Sales Analysis

## 📋 Dashboard Features

* Interactive Month Filter
* Interactive Day Filter
* KPI Cards
* Sales & Footfall Analysis
* Top 5 Products Analysis
* Category Distribution
* Store Performance Comparison
